## Review Summary

This document contains the full UX review feedback received from a USFS domain expert. The critical findings have been incorporated into the updated feature spec (v2.0).

### Key Changes Made Based on Review

| Original | Updated | Rationale |
|----------|---------|-----------|
| "Intel Probe" | **"Site Analysis"** | USFS is a science-based civilian agency, not military |
| "Research Mission" | **"Run Analysis"** | Direct, clear action language |
| "Intelligence Briefing" | **"Analysis Report"** | Professional, USFS-aligned |
| "Probe Target" | **"Selected Feature"** | Neutral terminology |
| "Context Anchor" | **"Feature Details"** | Plain language |

### Chip Label Rewrites

| Original | Updated |
|----------|---------|
| Verify USFS ID | Check NFS database |
| Prior incidents | Maintenance history |
| Access/haul | Log haul routes |
| Adjacent values | Nearby resources |
| Soil damage | Erosion risk |
| Hydrology impact | Drinking water concerns |
| Salvage window | Time left to salvage |

### Missing Chips Added

- Wilderness boundary proximity
- Tribal consultation requirements
- Pending litigation or appeals
- Invasive species concerns
