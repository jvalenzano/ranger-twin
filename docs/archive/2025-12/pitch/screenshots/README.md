# Screenshots Directory

This directory contains annotated screenshots for pitch decks.

## Naming Convention

```
[agent]-[feature]-[date].png
```

Examples:
- `burn-analyst-reasoning-chain-2025-12-28.png`
- `coordinator-briefing-synthesis-2025-12-28.png`
- `nepa-advisor-fsm-citation-2025-12-28.png`

## Capture Settings

| Setting | Value |
|---------|-------|
| Browser | Chrome (latest) |
| Viewport | 1920 x 1080 |
| Theme | Dark mode enabled |
| Accordions | Expanded |
| DevTools | Hidden |

## Annotation Guidelines

After capturing raw screenshots:

1. Import into Figma or Canva
2. Add callout arrows using brand colors
3. Use `Inter` font for annotations
4. Export at 2x resolution for print quality
5. Save both raw and annotated versions

## Storage

- Raw screenshots: `screenshots/raw/`
- Annotated versions: `screenshots/annotated/`
